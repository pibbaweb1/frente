# PrintMaster - Sistema de Impressão

Sistema profissional para impressão de múltiplas cópias em uma única folha, com suporte a frente e verso.

## Funcionalidades

- ✅ Upload de documentos (frente e verso)
- ✅ Configuração de dimensões personalizadas
- ✅ Layout de múltiplas cópias (até 10x10)
- ✅ Centralização automática
- ✅ Pré-visualização em tempo real
- ✅ Geração de PDF
- ✅ Interface responsiva
- ✅ Validação de segurança

## Tecnologias Utilizadas

- HTML5
- CSS3 (Grid, Flexbox, Variáveis CSS)
- JavaScript (ES6+)
- PHP 7.4+
- jsPDF (para geração de PDFs)
